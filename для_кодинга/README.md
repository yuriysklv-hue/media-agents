# media-agents — старт

## Что делать

1. Создай репозиторий `media-agents` на GitHub.
2. Склонируй его локально.
3. Скопируй `ТЗ_Новостной_пайплайн.md` в корень репозитория.
4. Открой папку в Claude Code.
5. Дай команду: **«Реализуй спецификацию из ТЗ_Новостной_пайплайн.md»**.

## Перед запуском

- Получи Zhipu API ключ: [open.bigmodel.cn](https://open.bigmodel.cn) (GLM-4-Flash бесплатен). См. раздел 14 в ТЗ.
- DeepSeek API ключ у тебя уже есть.
- Создай GitHub PAT с правами `repo` + `pull-requests` для Publisher.
