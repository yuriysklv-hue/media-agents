Дополни front-matter материала.

Правила:
- Slug: транслит русского заголовка → kebab-case → латиница → до 60 символов.
  Для брендов — устоявшееся написание: yandex, vk, google, meta.
- Description: 140-160 символов, с ключевым словом, без кликбейта.
- Category: одна из {categories_list}. Дефолт для западных новостей: adtech-world.
- Geo: массив из {geo_values} (РФ, МИР, АЗИЯ). Для западных новостей: ["МИР"].
- Tags: 3-7 тегов из контролируемого словаря: {tags_list}.
  Если ни один не подходит — можешь предложить новый, пометив [new].
- social_title: короткий заголовок для соцсетей, до 100 символов, без кликбейта.

Материал:
Title: {title}
Body (первые 500 символов): {body_preview}
Текущий front-matter: {current_frontmatter}

Верни строго JSON:
{
  "slug": "...",
  "description": "...",
  "category": "...",
  "geo": ["..."],
  "tags": ["..."],
  "social_title": "..."
}
